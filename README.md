# GreenCity - Database

Questo repository contiene il database relazionale per il progetto **GreenCity**, un portale ambientale che consente a cittadini, enti pubblici e aziende di monitorare e partecipare a iniziative ecologiche.

## Contenuto

- `sql/schema.sql`: Script SQL per creare il database.
- `doc/relazione_database.md`: Documentazione delle tabelle.

## Requisiti

- MySQL o PostgreSQL
- Un client SQL come DBeaver, phpMyAdmin o VS Code (con estensione SQL)
